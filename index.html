Installing Postgres SQL

By the end of the class, we want to complete installation of the Postgres server and be able to see it run in your local machine.

The instructions has been uploaded in Activity 1, under the Resources folder. Both Mac and Windows are included.

Notes:
  • There are a lot of ways to install a SQL server. Schools teach using Graphical User Interface (GUI) installer, but on a typical basis, we install it via command line.
    • Using Homebrew (MacOS) or a Linux command to install Postgres SQL.
    • On a typical setting, many servers within the cloud, and we don't install a GUI installer because it is unnecessary overhead in terms of security and resources.
    • Thus, don't be alarmed if you find that our installation patterns are different between work and school. They are the same engine, just different ways and representation.
  • I'm more used to MySQL. But ever since Oracle bought Sun Microsystems (MySQL), many companies are pivoting away from MySQL.
    • In your career, you will learn about different database engines if your work involves working with different systems.
