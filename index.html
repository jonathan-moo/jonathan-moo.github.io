JobTrack

This helps you in your career search. We have some presentation slides, and there are some instructors for you to use the platform.

  • STEP 1: Sign in to JobTrack at https://jobtrack.trilogyed.com using BCS credentials
  • STEP 2: Install the Chrome extension from https://chrome.google.com/webstore/detail/jobtrack-for-chrome/ojhnaheochknmclpabkpialdgkkcboni
  • Step 3: Find three jobs you think you’ll be interested in:
    • Visit Dice.com, Indeed.com, Angel.co, or go to the career website of a company you’re interested in
  • If Chrome extension installed, open it on the job posting and enter the details
  • If no Chrome extension, copy the URL, switch to JobTrack and enter the URL, Company Name and Job Title
  • When ready to apply to jobs, return to JobTrack and use these jobs as a starting point
  • *Career Director will discuss these jobs on coaching call to identify what your ideal job is and how to attain it*

01-Stu_CitiBike: Group effort

So we'll be plotting a graph using Citi Bike data(https://gbfs.citibikenyc.com/gbfs/en/station_information.json). This is not GeoJSON data, so you'll have to plot here.

There are 2 versions of this Citi Bike activity:

  • Basic: Using plain markers only. There are no plugins.
  • Advanced: We're using a plugin: https://github.com/coryasilva/Leaflet.ExtraMarkers
    • In addition to the data endpoint for station information, we need a station status endpoint: https://gbfs.citibikenyc.com/gbfs/en/station_status.json
    • Deploying this on Github Pages and using it as a portfolio could be impressive.

We have to start a HTTP server to run the code: python -m http.server
