02-Evr_Python_to_JavaScript: Please try on your own!

How to declare variables.

  • Use var: var x;
    • You'll have to declare the data type before declaration.
  • Use template literals: Start the string not with quotes, but with backquotes (I think they call it backticks in your instructions)`${variable}`
    • If you use quotes or double quotes, template literals cannot work.
    • Quotes or double quotes cannot run over the next line. You'll experience errors. But backticks can do multiple lines without breaking.

Conditionals

  • Use === instead of ==
    • == still works, but it doesn't check the type of the data
    • Example, "2" == 2 will generate a True. But "2"===2 will generate a False.
    • Instead of and in Python, use && in Javascript.
    • Instead of or in Python, use || in Javascript.
      • Javascript comparisons are more inline with traditional computer science languages (C++, Java).
    • Javascript has no indentation. You use curly brackets to wrap code instead of indentation for hierarchy.
      • 
if(x>y){
// This (x>y) runs first


	if(y < q){
		// This (x < q) runs within the first curly brackets.
	}
}

04-Evr_JavaScript_Arrays

  • Use var x = []; in Javascript. Unlike Python list, we've to declare the data type.
  • List in Python = Arrays in Javascript. Just different terminologies.
  • Use x.push(); to insert into your arrays. Python use x.append()
  • Use x.slice(1,3); which means element 1 to 3. Remember, the index starts from zero.
  • Javascript join list: x.join(" ");
    • Python is strange in terms of joining list into a sentence: " ".join(x)
