Purpose of this activity

Modeling databases is one of the most important things to do as a Data Engineer. If you model wrongly, your queries may give inaccurate results.

  1. Analyze the data and its schema.
  2. Model your database.
  3. Extract them into your Python scripts.
  4. Transform them into proper data types within the tables with your scripts.
  5. Load them into the tables.

Before we begin the activity:

  • Browse customer_data.csv
  • Browse customer_data.json
  • Create the database with: CREATE DATABASE customer_db;
  • Create the `customer_name` table:
    • CREATE TABLE customer_name ( id INT PRIMARY KEY, first_name TEXT, last_name TEXT );
  • Create the `customer_location` table:
    • CREATE TABLE customer_location ( id INT PRIMARY KEY, address TEXT, us_state TEXT );
  • Continue with our Jupyter Notebook Activity.
  • Perform a JOIN statement in SQL
    • SELECT customer_name.id, customer_name.first_name, customer_name.last_name, customer_location.address, customer_location.us_state FROM customer_name JOIN customer_location ON customer_name.id = customer_location.id;
